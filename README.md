# 英単語テストアプリ

React (Vite) で構築された英単語暗記用アプリです。選択した単語帳の中から出題し、英→日と日→英の2つのモードで瞬発力を鍛えることができます。マイク音量で「答えを見る」ボタンが解放される仕組みや、復習用の PDF 出力・苦手単語モードなど学習補助機能も備えています。

## 主な機能

- **単語帳の選択**: `src/data` 以下にある複数の単語リスト（英単語マスター 800 / English Gate / 難関高校英単語 / ターゲット1900 / Stock3000）を切り替え可能。
- **出題形式の切替**: 英単語→日本語、日本語→英単語をワンタッチで切替（設定はローカルストレージに保存）。
- **マイクモード**: 声を出して一定音量を超えると「答えを見る」ボタンが解放されるトレーニング機能。オフにすれば通常のボタン操作に戻ります。
- **出題範囲・ランダム設定**: ID 範囲指定、ランダム/順番通りの切り替え、制限時間を設定可能。
- **復習機能**: 間違えた問題は自動で記録され「苦手単語モード」で再出題。結果画面から復習モードを起動すると全問正解するまでシャッフル再挑戦できます。
- **PDF ダウンロード**: 間違えた単語だけをまとめた復習リストをワンクリックで生成。
- **音声読み上げ**: 英→日モードでは問題表示前に自動読み上げ、日→英モードでは答えを表示したタイミングで読み上げ可能。

## 動作環境

- Node.js 18 以上を推奨
- npm（同梱）

## セットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで表示されるローカル URL（デフォルトは `http://localhost:5173`）にアクセスしてください。

## ビルド

```bash
npm run build
```

`dist/` に静的ファイルが生成されます。`npm run preview` で本番ビルドをローカル確認できます。

## 使い方の流れ

1. ホーム画面で使用する単語帳を選択。
2. 「出題形式」で英→日 / 日→英を切り替え（初期値は英→日）。
3. マイクモードや出題範囲、制限時間、ランダム設定などを調整。
4. `テストを開始` を押して出題開始。スペースキーで答えを表示し、矢印キーで自己採点できます。
5. 結果画面から復習モードや PDF 出力、苦手単語モードへの遷移が可能。

## 開発メモ

- UI は Tailwind CSS を使用。`styles.css` で `@tailwind` 指令を読み込んでいます。
- マイク計測や音声読み上げはブラウザ API（`navigator.mediaDevices`, `speechSynthesis`）を利用しているため、HTTPS 環境や対応ブラウザでのテストを推奨します。
- 苦手単語や出題形式、マイクモード設定は `localStorage` に保存しており、ブラウザを閉じても状態が維持されます。

この README はアプリの機能とセットアップ手順をすぐ把握できるようにまとめています。より詳しい実装は `src/App.jsx` を参照してください。
